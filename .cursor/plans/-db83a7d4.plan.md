# PythonAnywhere 설정 완료

## 발견된 문제들

1. ❌ `rest_framework_simplejwt` 패키지 미설치
2. ❌ ALLOWED_HOSTS에 PythonAnywhere 도메인 미등록
3. ❌ 마이그레이션 미실행 가능성

## 해결 단계

### 1. ALLOWED_HOSTS 설정 (로컬에서 수정 후 push)

`config/settings.py` 파일 수정:

```python
ALLOWED_HOSTS = ['172.20.33.191', '127.0.0.1', 'localhost', '10.110.192.1', 'youngbin.pythonanywhere.com']
```

### 2. GitHub에 Push

```bash
git add config/settings.py
git commit -m "Add PythonAnywhere domain to ALLOWED_HOSTS"
git push origin main
```

### 3. PythonAnywhere Bash 콘솔에서 실행

```bash
# 프로젝트 디렉토리로 이동
cd ~/meongnyang

# GitHub에서 최신 코드 가져오기
git pull origin main

# 가상환경 활성화 (Web 탭에서 확인한 경로)
source /home/youngbin/virtualenvs/meongnyang-venv/bin/activate

# 패키지 설치
pip install -r requirements.txt

# 마이그레이션 실행
python manage.py makemigrations
python manage.py migrate

# 정적 파일 수집 (필요시)
python manage.py collectstatic --noinput
```

### 4. 웹 앱 재시작

Web 탭에서 "Reload" 버튼 클릭

## 실행 순서

1. 로컬에서 settings.py 수정
2. GitHub push
3. PythonAnywhere에서 명령어 실행
4. 웹 앱 재시작
